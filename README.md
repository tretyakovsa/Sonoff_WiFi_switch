# Upload instruction

## Specification .bin files
  -  Module: Generic ESP8266 Module
  -  Flash Size: 1M(256K SPIFFS)
  -  CPU Frequency: 80Mhz
  -  Flash Mode: dio
  -  Flash Frequency: 40Mhz
  -  Upload Speed: 115200

## Features
- NTP
- DDNS
- MQTT
- TIMERS
- DHT11, DHT22, AM2302, RHT03, D18B20
- A0
- Tachе
- RELAY
- JALOUSIE
- WS2811/WS2812/NeoPixel LEDs
- MOTION
- RC Switch
